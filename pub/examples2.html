Advanced Timeline Library

Create your own timeline!

Using the buttons below you can create your own timeline. Use the 'Back to Sample Timeline' link at the bottom of the page to return to the main example.

Instructions

  • Start by clicking on the 'Create Timeline' button. Note: this button can only be pressed once
  • You can add more main points to your timeline by clicking the 'Create Point' button. Note: this button will only work when not zoomed into a subdivision
  • Click on a main point to view its information card
  • Clicking on a subdivision (purple line) will zoom you into the line
  • When zoomed into a subdivision, you can add subpoints to it using the 'Create SubPoint' button. Note: this button will only work when zoomed into a subdivision
  • Click on the subpoint to view itss information card
  • Clicking on the subdivision (purple line) will zoom you out of the line and you can create the more main points
  • Repeat the steps above as many times as you'd like
