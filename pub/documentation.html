Getting Started
A quick overview of how to use the Advanced Timeline Library
Step 1: Install jQuery

This library requires the external library jQuery to function. You can install it using the instructions on their webpage (Install jQuery) or you can include the script in your html file using the code below.

                <script defer src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
Step 2: Install the Advanced Timeline Library

Click the following button to download the library's css and javascript files Download .

Include the css file in your html file using the code below.

                <link rel='stylesheet' type='text/css' href='AdvancedTimeline.css'></link>

Include the javascript file in your html file using the code below.

                <script defer type="text/javascript" src="AdvancedTimeline.js"></script>
Step 3: Initialize an Advanced Timeline

Create a javascript file and include the script in your html file using the code below.

                <script defer type="text/javascript" src="example.js"></script>

Call the constructor function to create an AdvancedTimeline object.

                const timeline = new AdvancedTimeline();
Step 4: Create a Timeline

Use the makeTimeline method to create a timeline. Provide the id of the Document Object Model (DOM) element in which you would like to create the timeline as the first parameter. Specify the value of the starting and ending points.

Optionally, you can specify a description of these points. This function returns the id of the starting and ending points. 

                const [startPoint, endPoint] = timeline.makeTimeline('#example1', false, 'white',  "Start", "End", 'Starting Point', 'EndPoint')
Step 5: Create a Point

Use the makeNewPoint method to create a point. Provide the value of the point as the first parameter. Optionally, you can specify a description of the point. This function returns the point's id. 

                const middlePoint = timeline.makeNewPoint("Middle", 'Middle Point')
Step 6: Create a Subpoint

Use the makeSubPoint method to create a subpoint for a point. Provide the id of the point in which you would like to create the sub point as the first parameter. Specify the value of the subpoint and optionally a description for it. This function returns the subpoint's id. 

                const subPoint = timeline.makeSubPoint(startPoint, "Point 1", "Point 1 Information")
Step 7: Add Text to a Subpoint's Information Card

Use the addTextElementToSubPoint method to add text to the subPoint's information card. Provide the id of the subpoint in which you would like to add the text as the first parameter. Specify the value of the text. You can also add images, audio, and video elements. 

                timeline.addTextElementToSubPoint(subPoint, 'This is the first subpoint')
API Methods
AdvancedTimeline()

The advanced timeline constructor. It returns an AdvancedTimeline Object.

AdvancedTimeline Object Properties

Property Default Type Description
activeSubDivion null String The id of the subDivision DOM element that is currently zoomed in
horizontal false Boolean When this value is true the timeline is horizontal. If false it is vertical
pointEndElement null Object The DOM element of the last point in the timeline
timeline null Object A javascript object that holds the points, subdivisions, and background colour that are displayed in the browser
timelineElement null Object The DOM element of the timeline
timelineWrapperElement null Object The parent DOM element of the timeline
Note: Styling the Timeline

The functions below all have a parameter that allows you to style the timeline's elements. The parameter is an object of css properties to be applied to the specified element. This object should be formatted as seen in the code below. Changing the height and width properties of points and subdivisons is not recommeded as this may conflict with the calculations used for zooming in and out of subdivisions. 

                const style = {
    'color': 'blue',
    'font-size': '15px'
};

Some elements have a box-shadow property on hover. The colour of this can be changed by including the variable '--box-shadow-color' in the style object.

                const style = {
    '--box-shadow-color': '#29d198'
};

To remove an existing css property, add it to the style object with an empty string.

                const style = {
    'color': '',
    'font-size': ''
};

AdvancedTimeline.makeTimeline()

Creates a timeline element in the DOM with a starting point, the starting point's subdivision, and an ending point. Ending points do not have a subdivision. It returns the ids of the starting and ending points' DOM elements.

Parameter Default Type Description
domElement 'body' String The id of the DOM element in which the timeline element should be added to
horizontal false Boolean If this value is true the timeline will be horizontal. If false it will be vertical
backgroundColor 'white' String The background colour of the timeline. Can be specified in css' accepted color formats (HEX, RGB, etc.)
startTitle 'Start' String The value of the timeline's starting point
endTitle 'End' String The value of the timeline's ending point
startInfo 'N/A' String The description of the timeline's starting point
endInfo 'N/A' String The description of the timeline's ending point
startPointStyle {} Object

An object of css properties to be applied to the starting point
startDivisionStyle {} Object

An object of css properties to be applied to the starting subdivision
endPointStyle {} Object

An object of css properties to be applied to the ending point

AdvancedTimeline.makeNewPoint()

Creates a point element in the DOM. It returns the id of the point's DOM element.

Parameter Default Type Description
pointTitle 'Point' String The value of the point
info 'No Information' String The description of the point
pointStyle {} Object

An object of css properties to be applied to the point
divisionStyle {} Object

An object of css properties to be applied to the point's subdivision

AdvancedTimeline.makeSubPoint()

Creates a subpoint element in a point's subdivision DOM element. It returns the id of the subpoint's DOM element. Note: subpoints cannot be added to the ending point in the timeline as it does not have a subdivision.

Parameter Default Type Description
pointId(required) - String The id of the point to create the subpoint for
frontInfo 'Click to see more information' String The description on the front of the subpoint's information card
backHeader '' String The header on the back of the subpoint's information card
style {} Object

An object of css properties to be applied to the subpoint

AdvancedTimeline.addTextElementToSubPoint()

Adds a text element to a subpoint's information card. It does not have a return value.

Parameter Default Type Description
subPointId(required) - String The id of the subpoint to add the text element to
text '' String The text to be displayed
top '' Number The top position of the text element in pixels
left '' Number The left position of the text element in pixels
right '' Number The right position of the text element in pixels
bottom '' Number The bottom position of the text element in pixels
styles {} Object

An object of css properties to be applied to the text element

AdvancedTimeline.addImageElementToSubPoint()

Adds an image element to a subpoint's information card. It does not have a return value.

Parameter Default Type Description
subPointId(required) - String The id of the subpoint to add the image element to
imageSrc(required) - String The path to the image
top '' Number The top position of the image element in pixels
left '' Number The left position of the image element in pixels
right '' Number The right position of the image element in pixels
bottom '' Number The bottom position of the image element in pixels
styles {} Object

An object of css properties to be applied to the image element

AdvancedTimeline.addAudioToSubPoint()

Adds an audio element to a subpoint's information card. It does not have a return value.

Parameter Default Type Description
subPointId(required) - String The id of the subpoint to add the audio element to
audioSrc(required) - String The path to the audio file. Only mp3 files are accepted
audioTitle '' String The value of the audio's label
top '' Number The top position of the audio element in pixels
left '' Number The left position of the audio element in pixels
right '' Number The right position of the audio element in pixels
bottom '' Number The bottom position of the audio element in pixels
styles {} Object

An object of css properties to be applied to the audio element's label

AdvancedTimeline.addVideoElementToSubPoint()

Adds a video element to a subpoint's information card. It does not have a return value.

Parameter Default Type Description
subPointId(required) - String The id of the subpoint to add the video element to
videoSrc(required) - String The path to the video. Only mp4 files or links to videos on external webpages are accepted
webpageSrc false Boolean If the videoSrc parameter is a link to a video on an external webpage, such as youtube, then this should be true. If the videoSrc parameter is an internal path then it should be false
top '' Number The top position of the video element in pixels
left '' Number The left position of the video element in pixels
right '' Number The right position of the video element in pixels
bottom '' Number The bottom position of the video element in pixels
styles {} Object

An object of css properties to be applied to the video element
